Adding Signature To Multiple Reports In Odoo

Description

This module extends the functionality of Odoo 16 by adding a signature feature to the reports of Sale Orders, Quotations, Invoices, Purchase Orders, and Requests For Quotation. It allows users to update signatures and display them in these reports, with the option of positioning the signatures on the left.

Configuration
Configure Signature Module Parameters
Go to the menu "Settings" - "General" - "Signature Parameters".
Here you can select the reports, position and more signature module options.
Selecting Company Signture
After configuration in settings go to the "Users & Companies" - "Companies" - "Select the company" - "Replace or add your signature".
Selecting Individual User Signature
After configuration in settings go to the "Users & Companies" - "Users" - "Select the user" - "Replace or add your signature".
